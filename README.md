# Arc Scanner Public Distribution

This repository hosts public distribution assets for Arc Scanner.

## Windows Release Bundle

Current bundle path:
- `releases/win-x64/`

Included:
- `ArcScanner.Desktop.Wpf.exe`
- runtime dependencies (`.dll`, `.json`, native libraries)
- `SHA256SUMS.txt`

Runtime requirement:
- Windows x64 with .NET 8 Desktop Runtime installed.

Source implementation is maintained in the private client repository.
